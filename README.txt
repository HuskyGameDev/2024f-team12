Moreau

Moreau is a story-driven 2D pixel-art game where you need to play as a detective in a near future dystopia.

Controls:
WASD / arrow keys to move
E to interact with surroundings
SPACE to skip/continue in dialogue or other interactions

Installation:
After downloading the .zip folder, extract ALL files, not just the executable. Then, you can launch as normal.
Windows defender will most likely create warnings
